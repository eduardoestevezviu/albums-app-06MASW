# Albums App

Práctica 2 de la asignatura **06MASW - Desarrollo de Aplicaciones Móviles**.

Aplicación móvil desarrollada con **React Native (Expo)** para catalogar
álbumes de música. Permite crear, listar, editar y borrar álbumes, que se
guardan localmente mediante **SQLite**.

## Cómo ejecutar

1. Instalar dependencias:

```bash
   npm install
```

2. Arrancar la app:

```bash
   npx expo start
```

3. Escanear el código QR con la app **Expo Go** (en iPhone, con la cámara).

## Tecnologías

- React Native con Expo
- Expo Router (Stack y Tabs)
- SQLite (`expo-sqlite`)
- TypeScript
